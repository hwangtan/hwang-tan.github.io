## About

<p>
자바스크립트를 주력 언어로 서비스를 만들고 있습니다. 1년전 부터 Typescript 를 주로 사용 하고 있으며 UI 를 그리는데 라이브러리는 Angular(ver.1) 6개월, React 2년 6개월 이상의 경험을 가지고 있습니다.
